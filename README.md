# Marché BKR Parc

Site vitrine pour Marché BKR Parc, une épicerie halal située à Montréal.

## À propos
Ce projet présente les informations principales du commerce, ses produits, ses promotions et ses coordonnées.

## Contenu
- Page d’accueil
- Section produits
- Section promotions
- Section à propos
- Section contact avec carte et WhatsApp

## Utilisation
Ouvrir `index.html` dans un navigateur web.
